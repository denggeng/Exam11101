# 出题者

出题者请按以下步骤修改文件:

1. 修改ExamDefine.go内容： 用于定义题目的输入和输出，单元测试文件的解析
2. 修改test_case内容： 用于验证答题者的单元测试文件
3. 修改ExamXXYY.go内容： 此文件为答题人作答的地方，请出题人在此处通过注释描述好题目内容

# 答题者

答题者请按以下步骤答题

1. 答题者修改ExamXXYY.go文件名：其中XX代表第几周，YY代表第几号题目，例如第二周第一题'Exam0201'；
2. 在1描述的go文件（ExamXXYY.go）中编写自己的代码
3. 自测：执行app_test.go
4. 提交文件到ftp自己的根目录。（只提交被编辑的ExamXXYY.go那个文件即可）
